# Fieldguard AI Platform

An AI-driven field data governance and intelligent decision-making platform designed for infrastructure, energy, and asset management industries.  

The platform focuses on field inspection workflows, data governance, AI-assisted analysis, and operational decision support, helping enterprises improve data standardisation, inspection efficiency, and asset management capabilities.  

## Core Features
- Field inspection and task management  
- AI-assisted anomaly detection  
- Data governance and visual analytics  
- Intelligent operational decision support  
- Interactive dashboard and workflow management  

## Tech Stack
- HTML / CSS / JavaScript  
- Axure RP  
- AI-assisted product design workflow
